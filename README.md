**Eye Gaze Tracking** is a Python-based computer vision project that detects the direction of a user's gaze using a webcam. The system uses OpenCV and MediaPipe Face Mesh to identify facial landmarks and extract eye regions in real time. By analyzing the distribution of pixels in the eye area, the program determines whether the user is looking left, right, or straight ahead. This project demonstrates basic eye-tracking concepts and real-time video processing for potential use in human–computer interaction systems.

**Eye Gaze Tracking** — это проект компьютерного зрения на Python, который определяет направление взгляда пользователя с помощью веб-камеры. Система использует библиотеки OpenCV и MediaPipe Face Mesh для обнаружения ключевых точек лица и выделения области глаз в реальном времени. Анализируя распределение пикселей в области глаза, программа определяет, смотрит ли пользователь влево, вправо или прямо. Проект демонстрирует базовые принципы отслеживания взгляда и обработки видео в реальном времени для применения в системах взаимодействия человека и компьютера.
